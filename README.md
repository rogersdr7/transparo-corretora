## Como publicar na Vercel
1. Faça upload deste ZIP em vercel.com → *Upload Project*.
2. Após o deploy, adicione o seu domínio `transparocorretora.com.br` nas configurações do projeto.
3. Crie os registros DNS na HostGator:
   - CNAME `www` → `cname.vercel-dns.com`
   - A `@` → `76.76.21.21` (ou o IP indicado pela Vercel)
